Desenvolvimento desse site

Descrição


Esse portfólio foi desenvolvido através do curso de HTML e CSS. Com ele foi possível aprender diversos truques de CSS e algumas formas de organização de código iniciais.

Tive a liberdade, durante o curso, de editar da forma que achei melhor todo o código e customizar da forma que achei interessante. Com isso, adicionei o bootstrap (apenas para estilizar e mostrar a integração), assim como, Font Awesome para alguns ícones.

Mesmo sendo um projeto básico, mostro por aqui a proatividade e olhar crítico para a confecção de um pequeno projeto.
